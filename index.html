Requirements and privacy

Requirements

  • Android 8.0+ (API 26+) with internet access for live data.
  • Android Studio, JDK 11, and Android SDK API 36 to build from source.
  • You only need an API key if you turn on AI summaries.

Permissions

  • INTERNET for compound data, descriptions, and release checks.
  • POST_NOTIFICATIONS for optional update alerts.
  • WRITE_EXTERNAL_STORAGE only for legacy export support on Android 9 and below.

Privacy

The app has no account system, analytics, tracking, or ChemSearch backend. Favorites, recents, cache, settings, and encrypted API keys stay on your device. Requests go to the data source or AI provider you choose.

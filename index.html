A toolkit arranged as connected lanes instead of 11 separate boxes.

The tools are grouped into larger merged panels so the section reads like a system: model viewing and reference on one side, calculation workflows on the other.

Reference and model tools

The part you reach for when you need to inspect or translate a compound quickly.

Custom 3D Molecule Viewer

Open any .mol or .sdf file from your device and inspect it in the same native viewer used for PubChem compounds.

SMILES Visualizer

Paste a SMILES string and jump to a rendered 2D structure plus a 3D model when PubChem has a matching record.

Isomer Finder

Enter a molecular formula to retrieve structural isomers, or jump in directly from a compound formula on the result page.

Oxidation State Finder

Handle common oxidation-state edge cases including peroxides, superoxides, interhalogens, and charged species.

Molar Mass Calculator

Work through nested formulas, hydrates, and formula syntax without simplifying them by hand first.

Requirements and privacy

Requirements

  • Android 8.0+ (API 26+) with internet access for live data.
  • Android Studio Hedgehog or newer to build from source.
  • No API key required unless you enable optional AI summaries.

Permissions

  • INTERNET for compound data, descriptions, and release checks.
  • POST_NOTIFICATIONS for optional update alerts.
  • WRITE_EXTERNAL_STORAGE only for legacy export support on Android 9 and below.

Privacy

The app does not require an account. Favorites, recents, cache, and any API keys stay on your device, and requests go directly to the upstream data source or provider you enable.

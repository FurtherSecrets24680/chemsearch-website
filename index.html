Install notes

  • Runs on Android 8.0 and newer.
  • Live lookup needs internet. Saved records and the built-in database can work offline.
  • Source builds use Android Studio and JDK 17+.
  • AI summaries need an API key.
  • No account, analytics, tracking, or ChemSearch backend.
  • Favorites, recents, cache, settings, and encrypted API keys stay on your device.

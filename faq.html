Install and setup

Getting started

What ChemSearch is, where it runs, and where to get the APK.

What is ChemSearch?

ChemSearch is a native Android chemistry reference app. It searches compounds, shows structures and safety data, copies identifiers, and includes chemistry tools.

Do I need an account to use it?

No. You can search compounds and use the tools without signing in.

What Android versions are supported?

ChemSearch targets Android 8.0 and newer, which is API level 26 and above.

Can I use ChemSearch on a tablet?

Yes. It runs on Android phones and tablets that meet the version requirement.

Where should I download the APK from?

Use the GitHub Releases page for the project. New APK builds are published there.

Does the app work fully offline?

Live compound search needs internet access because the app talks directly to PubChem, Wikipedia, GitHub Releases, and any AI provider you enable. The bundled Chemical Database works offline. Cached results, favorites, settings, and recent searches stay on the device.

Search and sources

Search and data

What you can search for, where the data comes from, and why some compound records have more detail than others.

What can I search by?

You can search by common name, IUPAC name, molecular formula, CAS number, or PubChem CID.

Which data sources does ChemSearch use?

PubChem PUG REST provides compound records, structures, autocomplete, identifiers, synonyms, and exports. PubChem PUG View provides safety annotations. Wikipedia can supply plain-language summaries. GitHub Releases handles update checks, and the bundled JSON files power the offline database.

Does CAS search work directly?

Yes. Paste the CAS number into search like any other query.

Why do some descriptions look different from one compound to another?

The source coverage varies. Some compounds have detailed PubChem summaries. Others fall back to a shorter Wikipedia or generated description.

Can I copy identifiers like SMILES, InChI, and CAS?

Yes. You can copy commonly reused identifiers from the result screen.

Does the app cache results?

Yes. Previously viewed compounds can load from the local cache, and recent searches are saved on the device.

Tools and outputs

Features and tools

What you can do after finding a compound, plus what lives in the Tools tab.

Can I view compounds in 3D?

Yes. Compound pages can switch from a 2D structure to an interactive 3D model when PubChem has the data.

Can I export structure files or images?

Yes. You can export structure data as SDF or PNG when those files are available.

How many built-in tools are included?

ChemSearch has 12 tools: Custom 3D Molecule Viewer, Molar Mass Calculator, Oxidation State Finder, SMILES Visualizer, Reaction Balancer, Isomer Finder, Limiting Reagent, Percent Yield, Reaction Scaling, Chemical Database, Dilution Calculator, and Ideal Gas Law.

What is included in the Chemical Database?

The offline database has 659 entries: 331 substances, 108 ions, 71 functional groups, and 149 reactions. Entries may include formulas, aliases, charges, uses, notes, source links, conditions, observations, and equations.

Does the app include elemental analysis and formula details?

Yes. Compound pages show formula details and elemental breakdowns when the source record supports them.

Are GHS hazards and signal words included?

Yes. ChemSearch pulls signal words, GHS pictograms, and hazard details from PubChem when available.

What happens when a record is missing some data?

The app shows whatever fields are available for that compound. Some sections are shorter when PubChem or another source has less data.

Keys and source builds

AI, privacy, and building from source

API keys, permissions, local storage, and source builds.

Do I need an API key for the app itself?

No. You only need an API key if you turn on AI summaries.

Which AI providers are supported?

AI summaries can use Google Gemini, Groq Cloud, OpenAI, OpenRouter, or Mistral AI. The rest of the app does not depend on them.

Does ChemSearch send my API key through its own backend?

No. Keys are encrypted locally with Android Keystore-backed storage, and requests go directly to the provider you choose.

What permissions does the app request?

The app asks for internet access, optional update notifications, and legacy export support on Android 9 and below.

Can I build ChemSearch from source?

Yes. Open the public repository in Android Studio. The current build notes call for Android Studio, JDK 11, and Android SDK API 36. For a debug APK, run ./gradlew assembleDebug.

Where should I report a bug or request a feature?

Use the GitHub issue tracker. That keeps bug reports and feature requests in the same place as the code.

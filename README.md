# ChemSearch Website

Landing page for the [**ChemSearch Android app**](https://github.com/FurtherSecrets24680/chemsearch-android).

The site mirrors the Android app's blue Material palette:

- Light: `#F8FAFC` background, `#FFFFFF` surfaces, `#2563EB` primary, `#DBEAFE` primary container.
- Dark: `#0F172A` background, `#1E293B` surfaces, `#3B82F6` primary, `#334155` outlines.

Content highlights are sourced from the Android app README and codebase: PubChem lookup, 2D/3D structures, GHS safety, 12 tools, 659 offline database entries, optional AI providers, privacy, permissions, and build requirements.
